# Active-Directory-homelab

## Project Overview

This project is a hands-on Active Directory lab built in Proxmox to develop skills relevant to Help Desk and IT Support roles.

## Objectives

- Learn Active Directory administration
- Practice Windows Server management
- Configure DNS and DHCP
- Troubleshoot common user issues
- Learn PowerShell automation

## Environment

### Hypervisor

- Proxmox VE
